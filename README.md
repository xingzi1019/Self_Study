## 📋 各阶段学习要点

### 🔵 已完成 · C 语言

- 数据类型与运算符、流程控制
- 函数与递归、数组与字符串
- **指针**、结构体与共用体
- 文件操作、动态内存管理

### 🔵 已完成 · C++

- 面向对象编程（类与对象）
- **STL 容器**（vector, list, map, set, stack, queue...）
- **STL 算法**（sort, find, transform...）
- **迭代器**、函数对象与 Lambda
- C++ 数据结构实现（链表、栈、队列、二叉树、堆、图）

### 🔵 已完成 · JavaSE

- 基础语法、面向对象
- 集合框架（List / Set / Map）
- 异常处理、IO 流
- 多线程、网络编程

### 🔴 当前阶段 · MySQL 数据库

**MySQL 数据库基础 → MySQL 数据库进阶**

| 课程             |   优先级    | 说明                     |
| ---------------- | :---------: | ------------------------ |
| MySQL 数据库基础 |  ⭐⭐⭐ 必学   | 数据库基本操作与 SQL     |
| MySQL 数据库进阶 | ⭐⭐ 高优先级 | 索引优化、事务、锁机制等 |
| MySQL 数据库高阶 |   ⭐ 中等    | 面试高频考点，冲击大厂   |
